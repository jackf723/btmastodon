# BTMastodon

BTMastodon is a mastodon client for the BTSpeak and BTBraille. It is written in python with much help from AI (specifically codex).
##installation
The easiest way to install this client on BTSpeak/BTBraille is to clone this repository into a directory on your computer, copy the .py files into a directory under home/btspeak/btcode/projects, then go into the btcode program and run **main.py to get started. You can also pin this file to your user menu to make it easier. from your project in btcode, hit dot 8 on **main.py and choose to either run it or pin it to your user menu for easier access.
